# Scripts Directory

This directory contains utility scripts for the programmatic SEO system.

## Scripts

- `deploy.js` - Deployment script to upload generated files to web server
- `validate-content.js` - Content validation and quality checks
- `validate-yaml.js` - YAML syntax validation
- `audit.js` - Content audit and performance reporting
- `bulk-update.js` - Bulk operations on content files

## Usage

All scripts are designed to be run from the project root:

```bash
# From /SEO directory
node scripts/validate-content.js
node scripts/deploy.js
node scripts/audit.js
```
